# LEVI'S Heritage Project Brief (Portfolio)

## 1. Project Overview
- Project Name: LEVI'S Archive Denim
- Type: Individual
- Duration: 2026.01.28 ~ 2026.02.02
- Role: Planning / Content Structuring / UI Design / Publishing (HTML/CSS/JS) / Interaction Implementation
- Deliverables: Multi-page website (Main + Sub pages: Heritage/Craft/Archive)

## 2. Background/Problem Definition
- Need for pages that emphasize brand heritage and narrative over commercial UI

## 3. Goals
- Visually express the materiality and temporality of denim/workwear
- Provide summary on the main page, in-depth narrative on sub pages
- Minimize commercial elements, focus on brand culture/archive

## 4. Target Audience
- Ages 20-40 interested in denim culture/workwear
- Users who value brand symbolism and history
- Users who want to understand brand values rather than shop

## 5. Core Concept/Keywords
- Heritage / Archive / Craft
- Indigo / Denim / Stitch
- Story-driven / Editorial layout

## 6. Information Architecture/Page Structure
- Home: Story summary, Read More for each section
- Heritage: Timeline from 1873 to present
- Craft: Detailed fabric/dyeing/sewing processes
- Archive: Lookbook/Texture/Cultural archive

## 7. Key Features/Interactions
- Hero Reveal + Typography entrance
- Stitch Line Draw (Section divider)
- Archive Hover Texture
- Read More Accent
- Parallax/Scroll Snap (Optional)

## 8. Design Direction
- Tone & Mood: Indigo/Heritage/Raw texture
- Colors: Indigo base + Red tab accent
- Typography: Oswald (Display) + IBM Plex Sans (Body)
- Layout: Editorial grid + Large images/Short copy

## 9. Tech Stack
- HTML5 / CSS3 / Vanilla JS

## 10. Accessibility/Responsive/Optimization
- prefers-reduced-motion support
- Recommended to disable excessive motion on mobile
- Image optimization and lazy-load (Incomplete)

## 11. Challenges/Differentiators
- Brand narrative-focused information architecture instead of commercial UI
- Emphasis on materiality through texture/stitch elements

## 12. Results/Lessons Learned
- Organized a narrative-driven information architecture and multi-page flow
- Built a consistent visual system with indigo tones, texture, and stitch motifs
- Designed low-motion interactions that maintain immersion and readability

## 13. Future Improvements
- Image optimization and lazy-load
- Accessibility audit (alt, aria-label)
- Responsive QA
- Code cleanup
