# LEVI'S Heritage 리디자인

Levi's 브랜드의 **헤리티지/아카이브/공정**을 중심으로, 커머셜 UI보다 서사를 강조한 멀티페이지 포트폴리오 프로젝트입니다.

## 프로젝트 개요
- 프로젝트명: LEVI'S Archive Denim
- 유형: 개인
- 기간: 2026.01.28 ~ 2026.02.02
- 역할: 기획 / 콘텐츠 구성 / UI 디자인 / 퍼블리싱(HTML/CSS/JS) / 인터랙션 구현
- 산출물: 메인 + 서브 페이지(heritage/craft/archive)

## 목표
- 브랜드 헤리티지와 시간성을 시각적으로 전달
- 메인 페이지에서 요약, 서브 페이지에서 깊이 있는 서사 제공
- 상업적 요소 최소화, 브랜드 문화/아카이브 중심

## 타겟
- 데님 문화/워크웨어에 관심 있는 20~40대
- 브랜드 상징/역사성을 중요하게 보는 사용자
- 쇼핑보다 브랜드 가치 이해를 원하는 사용자

## 키워드
- Heritage / Archive / Craft
- Indigo / Denim / Stitch
- Story-driven / Editorial layout

## 사이트맵
- Home (index.html)
- Heritage (heritage.html)
- Craft (craft.html)
- Archive (archive.html)

## 디자인 방향
- 톤&무드: 인디고/헤리티지/로우 텍스처
- 컬러: 인디고 베이스 + 레드 탭 포인트
- 타이포: Oswald(Display) + IBM Plex Sans(Body)
- 레이아웃: 큰 이미지 + 짧은 카피, 에디토리얼 그리드
